# Who Gave You Authority?

**Live Website:** https://brad-hobbswd.github.io/who-gave-you-authority

An immersive online reading experience for the book **Who Gave You Authority?** by **Bradley Hobbs**.

This website presents the complete manuscript in a carefully designed, fully responsive format that emphasizes readability, elegant typography, and a distraction free reading experience. Rather than functioning as a traditional promotional landing page, the website serves as a digital edition of the book, allowing readers to move naturally from the front matter through every chapter and into the concluding pages.

---

## ✨ Features

- Fully responsive design for desktop, tablet, and mobile devices
- Elegant typography designed for long form reading
- Complete online manuscript
- Professional front matter and legal pages
- Interactive table of contents
- Reading progress indicator
- Mobile friendly navigation
- Scroll to top functionality
- SEO optimized HTML structure
- Accessible semantic markup
- Fast loading static website
- GitHub Pages deployment

## 🎯 Project Goals

The purpose of this website is to provide readers with a beautiful online reading experience while faithfully presenting the complete manuscript of **Who Gave You Authority?**

The design emphasizes simplicity, readability, accessibility, and performance while keeping the focus on the message rather than unnecessary distractions.
